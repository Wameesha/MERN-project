### Backend Setup

1. Install backend dependencies:
    ```bash
    cd backend
    npm install
    ```
2. Configure environment variables (see `.env.example` for required settings).
3. Start the backend server:
    ```bash
    npm run dev
    ```

### Frontend Setup

1. Install frontend dependencies:
    ```bash
    cd ../frontend
    npm install
    ```
2. Start the frontend development server:
    ```bash
    npm run dev
    ```
